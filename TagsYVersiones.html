    • Comando para mostrar el log mas bonito y ordenado
    git log --all --graph --decorate --oneline

    Agregar alias al comando
    alias nombre="comando"

    git tag mostrar tags

    git tag -a <identificador> -m "<mensaje>" <UUID> crear un tag para identificar una version desde algun uuid

    git show-ref --tags mostar la referencia de los tags

    git push origin --tags sube los tags

    git tag -d <tag a borrar> elimina un tag de manera interna
    git push origin :refs/tags/<tag a borrar> elimina la tag en github
