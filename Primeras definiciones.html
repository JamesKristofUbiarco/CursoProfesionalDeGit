• Volver a inicio

    • git init Empieza un repositorio en la carpeta

    git add <filename> Preparar un archivo para mandarlo al repositorio staging

    git add. Agrega todos los cambios de la carpeta

    git commit -m "<mensaje>" Aplica cambios y agrega un mensaje de los cambios

    git status Revisa que archivos estan en staging

    git show Muestra todo el historial de los cambios del repositorio

    git log <mensaje> Muestra el historial completo de algun archivo

    git push Envia a un repositorio remoto los cambios

    git pull Trae todos los cambios del repositorio
